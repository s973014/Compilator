Постановка задачи

Константы – это неизменяемые величины, значения которых заданы заранее и не изменяются во время выполнения программы.

Для описания констант в языке Pascal используется служебное слово const.

Формат записи: const <имя_константы>:integer=значение;

Пример:

Целая десятичная константа – любое десятичное число без десятичной точки со знаком или без него:

const n:integer=12;

В связи с разработанной автоматной грамматикой G[‹START›] синтаксический анализатор (парсер) целочисленных констант будет считать верными следующие записи констант:

  • const n:integer=12;
  • const x : integer = -10;
  • const y: integer= +123;
