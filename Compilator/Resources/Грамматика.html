Разработка грамматики

Определим грамматику целочисленных констант языка Pascal G[‹START›] в нотации Хомского с продукциями P:

  • 1. <START> → 'const'<SPACE>
  • 2. <SPACE> → '_'<IDENTIFIER>
  • 3. <IDENTIFIER> → StartSymbols<IDREM>
  • 4. <IDREM> → Name<IDREM>
  • 5. <IDREM> → ':'<TYPE>
  • 6. <TYPE> → 'integer'<EQUAL>
  • 7. <EQUAL> → '='<NUMBER>
  • 8. <NUMBER> → '+'<UNSIGNEDNUM>
  • 9. <NUMBER> → '-'<UNSIGNEDNUM>
  • 10. <NUMBER> → Digit <NUMBERREM>
  • 11. <UNSIGNEDNUM> → Digit <NUMBERREM>
  • 12. <NUMBERREM> → Digit <NUMBERREM>
  • 13. <NUMBERREM> → ';'

Следующие множества определяют используемые символы:

  • StartSymbols → 'a' | 'b' | 'c' | ... | 'z' | 'A' | 'B' | 'C' | ... | 'Z' | '_'
  • Name → 'a' | 'b' | 'c' | ... | 'z' | 'A' | 'B' | 'C' | ... | 'Z' | '1' | '2' | '3' | '4' | '5' | '6' | '7' | '8' | '9' | '0' | '_'
  • Digit → '1' | '2' | '3' | '4' | '5' | '6' | '7' | '8' | '9' | '0'

Следуя формальному определению грамматики, представим G[<START>] ее составляющими:

  • Z = <START>
  • VT = { a, b, c, ..., z, A, B, C, ..., Z, _, =, +, -, ;, 0, 1, 2, ..., 9 }
  • VN = { <START>, <SPACE>, <IDENTIFIER>, <IDREM>, <TYPE>, <EQUAL>, <NUMBER>, <UNSIGNEDNUM>, <NUMBERREM> }
